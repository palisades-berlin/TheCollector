Help & FAQ

THE Collector helps you capture full pages, collect links, and find everything later. Start in the popup, then use History to filter and manage saved items.

1. Save content Open the popup and use Capture or URLs for fast saving.
2. Use Smart Save (Pro/Ultra) Choose Research, Interest, or Private to capture with that preset immediately.
3. Find and manage Open History to filter by domain, date, type, or profile (Pro/Ultra).

Frequently Asked Questions

Why is a feature missing?

Features are shown by tier. Basic includes core tools, Pro includes Basic plus advanced tools, and Ultra includes Pro plus advanced roadmap features as they ship.

How do Smart Save Profiles work?

In Pro/Ultra, choose Research, Interest, or Private in Capture to apply that preset and start capture immediately.

In v1 these are fixed presets. Editable profiles are planned for a later phase.

Where do I set defaults?

In Daily Essentials, set your tier, default Smart Save profile (Pro/Ultra), and theme. Export defaults are in Capture & Export.

Why did a page fail to capture?

Some browser-restricted pages cannot be captured. Switch to a regular website tab and try again.

How does queue capture finish?

Queue runs in the background for Pro/Ultra. When complete, History opens automatically with a queue result summary.

Where can I see profile usage?

In Pro/Ultra, you can review profile usage counts in History and in Feature Access on this Settings page.
