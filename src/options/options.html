Host Access

This build uses active-tab capture scope by default.

Help & FAQ

THE Collector helps you save useful pages fast and find them again when you need them. You can capture full pages, collect URLs, and organize everything in one place.

Don't worry, your data stays on your device. The extension does not upload your screenshots, URL lists, or settings. THE Collector is free forever.

Choose your feature surface level in Settings: Basic, Pro, or Ultra. Higher levels show more tools, but all levels are free and you can switch any time.

Frequently Asked Questions

How do I save a page quickly?

Open the popup and stay on Capture. Then use these steps:

  1. Click Capture Full Page.
  2. Wait for capture to finish.
  3. If capture is blocked, switch to a normal website tab.
  4. Try again from the same button.
Can I use a keyboard shortcut for capture?

Yes. Press Alt+Shift+P on a normal website tab to start capture quickly. If the page is restricted, switch to a regular website and try again.

How do I use Smart Save Profiles (Pro/Ultra)?

Smart Save gives you fixed presets: Research, Interest, and Private. Use these steps:

  1. Open Capture in the popup.
  2. Click Research, Interest, or Private.
  3. Capture starts right away with that preset.
  4. In Settings → Daily Essentials, pick your default Smart Save profile.

In this version, profiles are fixed presets.

How do I run Queue Capture (Pro/Ultra)?

Queue Capture helps you process multiple tabs in one run. Follow these steps:

  1. In Capture, click Queue Current or Queue Window.
  2. Click Run Queue.
  3. Wait for the run to finish.
  4. THE Collector opens Screenshots with queue results.
How do I find, filter, and manage screenshots?

Open Screenshots to review your saved captures. Click the domain field to open saved domain suggestions (with a screenshot count beside each), type to filter, and press Enter or click to apply.

Use the clear icon in the domain field to reset that filter quickly. You can also combine domain search with date, export type, and profile filters.

Watch for diagnostic hints like slow-run notes or the latest failure note when they appear.

How do I compare two screenshots?

Use the compare buttons in Screenshots to pick two items. Then open the compare view.

You will see Visual Diff overlays where green means added/brighter areas and red means removed/darker areas.

What can I do in Preview?

Open any screenshot to enter Preview. You can crop, blur, highlight, add text, draw rectangles, and place emoji markers.

Export options include PNG, JPG, and PDF (`Auto`, `A4`, `Letter`), plus quick presets like Email, Docs, and PDF Auto.

How do I use URL quick actions in the popup?

Go to popup URLs tab. You can copy all URLs, export TXT/CSV, send an email draft, clear the list, and restore the last cleared list.

Open URL Library when you want full views, filters, tags, notes, and bulk tools.

How do I use address bar shortcuts (Ultra)?

You can run actions from the browser address bar. Use these steps:

  1. Type tc in the address bar.
  2. Choose one: research, star, or queue.
  3. Press Enter to run the action.
How do I use Saved URL Views (Pro/Ultra)?

Open URL Library from the popup using the Library tab or Open URL Library. Switch views between All, Starred, Today, and By Domain. Views only change what you see, not what you saved.

How do I add tags to a URL (Pro/Ultra)?

In URL Library, click Tags on a URL row to expand controls. Add a tag from suggestions or type your own and click Add. You can save up to 10 tags per URL.

Use the Tag filter to show matching URLs. You can combine this with domain and date filters.

How does Auto-purge screenshot storage work?

Open Settings → Feature Access to control Enable Auto-purge oldest screenshots. It is on by default so capture can continue when storage reaches the screenshot limit.

Check Storage usage (for example 123/500 screenshots) anytime. If you turn Auto-purge off and reach the limit, new saves pause until you delete screenshots or enable Auto-purge again.

How do I add a note to a URL (Pro/Ultra)?

In URL Library, click Note on the URL row. Type your note inline and click Save. Notes are limited to 140 characters, and you can remove a note with Clear.

How do I use URL Bulk Actions (Pro/Ultra)?

Select one or more URL rows in URL Library. Then use bulk actions to Copy, Open, TXT/CSV export, or Delete. Use Select Visible when you want to select your current filtered list quickly.

How do I use URL Change Log?

In URL Library, open the Change Log tab to review snapshots. You can run Restore, Copy, TXT, or CSV actions from each entry. Press Esc or click Back to URL List to return.

How do Settings saves work?

Update your preferences, then click Save Settings. Changes apply after you save, including controls like Smart Revisit Nudges.

What do Smart Revisit Nudges and Weekly Value Report do?

Smart Revisit Nudges help you come back to important saved work. Weekly Value Report shows your local summary for captures, domains, URLs, and estimated time saved.

Why is a feature missing?

Your feature surface level controls what you see: Basic, Pro, or Ultra. Each level includes everything in the level below it. Switch levels any time in Settings, and remember all levels are free.

Where do I set my defaults?

Use Daily Essentials for your feature level, theme, and default Smart Save profile. Use Capture & Export and Downloads for export and file save behavior. Keep these defaults up to date so daily capture feels faster.

Any quick tips before I start?

If a page fails to capture, switch to a regular website tab and try again. The optional downloads permission only changes direct file saving, and fallback download still works.

Quick example: set level to Pro, choose Research, queue several tabs, then filter by profile in Screenshots to jump back into work.

Unsaved changes. Click Save Settings.
